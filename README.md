# web-final

## Quick Start

- install redis with 
``` js
sudo apt update
sudo apt install redis-server
```
- open a termainal and type to start redis server
``` js
sudo apt update
sudo apt install redis-server
```
- put MONGO_URL into .env
- yarn install
- open termainals and type 
``` js
yarn api-server
yarn socket-server
yarn start
```
